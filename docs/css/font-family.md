## 设置字体

  - font-family 属性设置文本的字体系列。

  - font-family 属性应该设置几个字体名称作为一种"后备"机制，如果浏览器不支持第一种字体，他将尝试下一种字体。

  - 注意: 如果字体系列的名称超过一个字，它必须用引号，如Font Family："宋体"。

  ``` css
    p{font-family:"Times New Roman", Times, serif;}
  ```
